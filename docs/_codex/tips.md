<ol class="codex-tips">
<li id="tip-1">
<p>You can disconnect devices quickly by using the <span class="nobr"><span class="accent"><b>Connect</b></span></span> tool. Just click and drag a line between the connected devices.</p>

</li>
<li id="tip-2">
<p>You can switch to the <span class="accent"><b>Connect</b></span> tool automatically by clicking and dragging a connector in Free Select mode.</p>

</li>
<li id="tip-3">
<p>You should keep every device accessible from at least one side, otherwise workers will not be able to reach them to perform repairs or maintenance.</p>

</li>
<li id="tip-4">
<p>Building devices on top of debris and materials will automatically create hauling tasks. Construction will not happen until the debris is moved.</p>

</li>
<li id="tip-5">
<p>You can quickly adjust task priorities for multiple objects by using the Manage Tasks tool.</p>

</li>
<li id="tip-6">
<p>Most type of storage can be configured to automatically haul compatible items. Look for the &quot;Auto Haul&quot; option in the storage settings.</p>

</li>
<li id="tip-7">
<p>Most types of processors can be configured to automatically process compatible items. Look for the &quot;Auto Haul&quot; option in the processor configuration.</p>

</li>
<li id="tip-8">
<p>Consider pausing time when reviewing or managing event notifications, otherwise new notifications can reorder the list and reset pagination.</p>

</li>
<li id="tip-9">
<p>You can use the <span class="nobr"><img class="inline-icon" src="/img/codex/sprites/glyphs/glyphs-icons-white/connect.png" alt="Connect"> <span class="accent"><b>Connect</b></span></span> tool to manage links between connectors and electrical objects by simply dragging a line with your mouse.</p>

</li>
<li id="tip-10">
<p>Don't rush to expand your ship; visit nearby planets and mine some raw materials first.</p>

</li>
<li id="tip-11">
<p>Reactors will not burn materials if solar panels can provide enough electricity.</p>

</li>
<li id="tip-12">
<p>Reactors will not produce more electricity than required to satisfy current energy demand.</p>

</li>
<li id="tip-13">
<p>If you want to scale your production, in addition to building new devices, try configuring existing ones to increase the total wanted amount, or changing the production type from &quot;Target Inventory&quot; to something else.</p>

</li>
<li id="tip-14">
<p>You can increase the Auto Refill Amount on most refillable devices, like the Matter Reactor. This will reduce the frequency of refills.</p>

</li>
<li id="tip-15">
<p>You can hold <kbd>Left Ctrl</kbd> to order a selected worker to perform a certain task.</p>

</li>
<li id="tip-16">
<p>You can hold <kbd>Left Shift</kbd> to order a selected worker to go to the position of your choice.</p>

</li>
<li id="tip-17">
<p>When ordering the placement of floor tiles, try holding various combinations of <kbd>Left Shift</kbd>, <kbd>Left Alt</kbd>, and <kbd>Left Ctrl</kbd> to switch into various placement modes. Some of these modes work with walls, and all of these modes work with the Planning tool.</p>

</li>
<li id="tip-18">
<p>There are two ways to make a circle in Planning mode. Begin with your mouse near the center of the tile to make a circle with an odd diameter, and begin at the lower-left corner of the tile to make a circle with an even diameter.</p>

</li>
<li id="tip-19">
<p>Turn off oxygen pumps and close all vents in case of fire. The Stasis Array has a built-in oxygen pump that can be turned off too. If that doesn't help, make a hole in the floor. A hole in the floor vents better than a hole in the wall.</p>

</li>
<li id="tip-20">
<p>This game is being developed live on Twitch. See how it's made at <a href="https://twitch.tv/dev_spajus" target="_blank" rel="noopener">twitch.tv/dev_spajus</a></p>

</li>
<li id="tip-21">
<p>The quickest way to build a new Connector is to click and drag a line from an existing Connector, then release it over an empty floor tile.</p>

</li>
<li id="tip-22">
<p>You can customize the color of floor lights.</p>

</li>
<li id="tip-23">
<p>See a problem with the game? Lack of balance? Boring gameplay? Difficult to understand a concept? Something inconsistent? Please use the &quot;Report a Bug&quot; button in the bottom right of the screen. Your report will automatically include a screenshot and your save file. Having a save file at the moment of the problem makes it much easier to reproduce and fix a problem, therefore this is the preferred way of reporting any issues.</p>

</li>
<li id="tip-24">
<p>You can create mods for this game without writing any code. Find out more about modding Stardeus at <a href="https://github.com/kodolinija/stardeus-mod-template" target="_blank" rel="noopener">github.com/kodolinija/stardeus-mod-template</a></p>

</li>
<li id="tip-25">
<p>For a more immersive gameplay, try the &quot;Fog of War&quot; option in Settings &gt; Video &gt; Ambient Light. You will have to build light sources to break through the darkness of space.</p>

</li>
<li id="tip-26">
<p>You can use <kbd>Left Ctrl</kbd> + numpad digit to remember the current camera position. Then press the numpad digit to jump there quickly.</p>

</li>
</ol>
