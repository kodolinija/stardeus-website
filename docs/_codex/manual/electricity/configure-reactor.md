<p><span class="nobr"><a href="/codex/database/objects/devices/matter-reactor-mini/">Matter Reactor</a></span> can be configured to burn almost any type of material. I recommend using <span class="nobr"><a href="/codex/database/obj/materials/scrap-metal/">Scrap Metal</a></span> or <span class="nobr"><a href="/codex/database/obj/materials/steel-plate/">Steel Plate</a></span>, as they produce a lot of electricity and are easy to obtain.</p>

<p>1. Press <span class="nobr"><kbd title="Mouse: Left Button">Left Button</kbd></span> on a <span class="nobr"><a href="/codex/database/objects/devices/matter-reactor-mini/">Matter Reactor</a></span>.<br>2. Select the desired fuel type in the UI.<br></p>
