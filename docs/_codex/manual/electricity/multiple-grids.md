<p>It is possible to have more than one electric grid. You can control devices and manipulate connections only if the Ship Computer is connected to the grid.</p>
<p>&gt;In most scenarios you should attempt to join multiple grids into one. In cases where you want to quickly disconnect part of the grid, you can build a Switch and route the electrical connections through it.</p>

<p>You can quickly find unmanaged electrical grids using the Electric Grid UI.</p>
