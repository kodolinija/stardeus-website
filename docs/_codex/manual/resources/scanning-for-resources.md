<p>A <span class="nobr"><a href="/codex/database/objects/devices/radar/">Radar</a></span> can scan the universe for specific resources and objects. Use the <span class="nobr"><span class="accent">Starmap</span></span> to configure resource scanning.</p>

<figure class="codex-video"><video src="/vid/codex/Production/WideScan.webm" controls loop muted playsinline preload="metadata"></video></figure>
<p>Radar also performs an automatic deep scan of the current sector. Deep scans can reveal several resource deposits per sector that targeted long-range scanning would not find.</p>

<figure class="codex-video"><video src="/vid/codex/Production/DeepScan.webm" controls loop muted playsinline preload="metadata"></video></figure>
<p>Unlock <span class="nobr"><a href="/codex/research/research/scanning/system/">System Scanning</a></span> research to scan for uncommon resource and object types.</p>

<p>Unlock <span class="nobr"><a href="/codex/research/research/scanning/extended/">Extended Scanning</a></span> research to scan for extremely rare resources like <span class="nobr"><a href="/codex/database/obj/materials/dark-matter/">Dark Matter</a></span> or <span class="nobr"><a href="/codex/database/obj/materials/edenium/">Edenium</a></span>.</p>

<p>Build additional <span class="nobr"><a href="/codex/database/objects/devices/radar/">Radar</a></span> or <span class="nobr"><a href="/codex/database/objects/devices/system-scanner/">System Scanner</a></span> to increase scanning speed.</p>
