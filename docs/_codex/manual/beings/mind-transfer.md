<p>The most important part of any robot is the <span class="accent"><b>Core</b></span>, which contains the bot's identity, skills, memories, experiences, and traits. If this part is <span class="accent"><b>Irreparably</b></span> damaged, the robot is considered <img class="inline-icon" src="/img/codex/sprites/glyphs/glyphs-icons-color/skull.png" alt="Skull"> <span class="accent"><b>Dead</b></span>.</p>
<p>&gt;For biological beings, this part is the <span class="accent"><b>Brain</b></span>.</p>

<p>For biological beings, some wounds can be <span class="accent"><b>Fatal</b></span>. However, if the <span class="accent"><b>Brain</b></span> remains intact, it can be <span class="accent"><b>transferred</b></span> into a different body.</p>

<p>Brains can even be swapped between <span class="accent"><b>different species</b></span>. For example, transplanting the brain of a <span class="accent"><b>Cat</b></span> into the body of a <span class="accent"><b>Human</b></span> will result in a human with the identity of the cat, who will behave like one.</p>

<p>While you cannot install a <span class="accent"><b>Brain</b></span> into a robot, and cannot replace the <span class="accent"><b>Brain</b></span> with a robotic <span class="accent"><b>Core</b></span> in a biological being, it is possible to perform a <span class="accent"><b>Mind Transfer</b></span> using the <span class="accent"><b>Replicator</b></span>.</p>

<p>Using a <span class="accent"><b>Core</b></span> removed from a defeated <span class="accent"><b>enemy</b></span> will <span class="accent"><b>resurrect</b></span> the enemy, but with the help of a <span class="accent"><b>Replicator</b></span>, you can <span class="accent"><b>overwrite</b></span> the enemy's <span class="accent"><b>identity</b></span> with one from a <span class="accent"><b>friendly</b></span> colonist <span class="accent"><b>Brain</b></span>.</p>
<p>&gt;This is a great way to expand your robotic workforce by giving a second life to your fallen biological colonists.</p>

<p>To use the <span class="accent"><b>Replicator</b></span>, both the <span class="accent"><b>Core</b></span> and the <span class="accent"><b>Brain</b></span> must first be removed from the bodies.</p>

<p>Biological organs can be <span class="accent"><b>harvested</b></span> at the <span class="accent"><b>Surgery Hub</b></span>.</p>
