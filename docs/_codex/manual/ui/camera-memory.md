<p>You can save the camera's position and zoom by pressing <span class="nobr"><kbd title="Keyboard: Left Control / Keyboard: Right Control">Left Control / Right Control</kbd></span> and a number on the numpad. Pressing the same number without holding <span class="nobr"><kbd title="Keyboard: Left Control / Keyboard: Right Control">Left Control / Right Control</kbd></span> will move the camera to the saved position.</p>
