<p><span class="accent"><b>Quick Search</b></span> will help you to perform various actions and find information. Open it with <span class="nobr"><kbd title="Keyboard: Tab">Tab</kbd></span> and start typing a query. You can:<br> - Switch into a <span class="accent"><b>Tool</b></span>.<br> - <span class="accent"><b>Build</b></span> a device.<br> - Find a <span class="accent"><b>Being</b></span>.<br> - Find a <span class="accent"><b>Research Node</b></span> or a <span class="accent"><b>Blueprint</b></span> in the <span class="accent"><b>Research Tree</b></span>.<br> - Find objects in the <span class="accent"><b>Starmap</b></span>.<br> - Find <span class="accent"><b>Codex</b></span> entries.<br> - Jump to a <span class="accent"><b>Coordinate</b></span> (e.g.: <span class="accent"><b>120 234</b></span>)</p>
