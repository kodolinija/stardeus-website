<p><span class="nobr"><a href="/codex/database/objects/devices/teleporter/">Teleporter</a></span> devices create one-way portals to other teleporters.</p>

<p>Link two teleporters together to create a two-way portal. Multiple teleporters can share the same destination.</p>

<p>Power consumption increases exponentially with the number of active connections.</p>

<p>Use the <span class="nobr"><kbd title="Keyboard: E">E</kbd></span> <span class="accent"><span class="accent"><b>Connect</b></span></span> tool to link or unlink teleporters.</p>
