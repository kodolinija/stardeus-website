<p>Each ship class has a maximum number of legally allowed Class Points. Each combat device (weapon, shield, thruster) consumes a certain amount of Class Points. Exceeding the limit results in negative consequences.</p>

<p>Loadout limits are defined by Class Points. Larger ships have higher-tier ship classes that allow more Class Points.</p>

<p>Every <span class="accent"><b>Ship Device</b></span> used in combat costs some Class Points.</p>

<p>Exceeding the allowed Class Points for your <span class="nobr"><a href="/codex/manual/ship-combat/ship-class/">Ship Class</a></span> is illegal and results in <span class="nobr"><a href="/codex/manual/ship-combat/outlaw/">Outlaw</a></span> status.</p>

<p>A notification will inform you when you violate the Class Points limit.</p>

<p>Being an <span class="nobr"><a href="/codex/manual/ship-combat/outlaw/">Outlaw</a></span> has drawbacks. Most traders will avoid you, and <span class="nobr"><a href="/codex/manual/ship-combat/law-enforcement/">Law Enforcement</a></span> officers will hunt you.</p>

<p>See: <span class="nobr"><span class="accent">Ship Computer → Apps → Ship Config</span></span> → Class Points</p>
