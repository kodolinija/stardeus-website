<p>Every ship has a <span class="accent"><span class="accent"><b>Ship Class</b></span></span> that defines allowed loadout limits.</p>

<p>If your ship no longer meets the class requirements, you must choose a new <span class="accent"><span class="accent"><b>Ship Class</b></span></span>.</p>

<p>Loadout limits are defined by <span class="nobr"><a href="/codex/manual/ship-combat/class-points/">Class Points</a></span>. Larger ships have higher-tier ship classes that allow more <span class="nobr"><a href="/codex/manual/ship-combat/class-points/">Class Points</a></span>.</p>
