<p>Click the ship combat widget to switch to the tactical view.</p>

<p>Press and hold the <span class="accent"><span class="accent"><b>Fire</b></span></span> button on any <span class="nobr"><a href="/codex/database/objects/devices/battle-station/">Battle Stations</a></span> that require manual control.</p>

<p>Monitor <span class="nobr"><a href="/codex/manual/ship-combat/flux/">Accumulated Flux</a></span> levels and use the Vent button when necessary to avoid <span class="nobr"><a href="/codex/manual/ship-combat/flux-venting/">Flux Overload</a></span>. Shields are disabled while venting flux.</p>

<p>Use <span class="nobr"><span class="accent">Ship Computer → Apps → Ship Config</span></span> to configure global weapon targeting or adjust the preferred targets of individual <span class="nobr"><a href="/codex/manual/ship-combat/battle-stations/">Battle Stations</a></span> to create your own combat strategy.</p>
