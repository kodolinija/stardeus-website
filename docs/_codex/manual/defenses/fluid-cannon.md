<p><span class="accent"><b>Fluid Cannons</b></span> can be fueled with almost any type of <span class="accent"><b>liquid</b></span>. When fueled with <span class="accent"><b>Water</b></span>, they not only deal <span class="accent"><b>light damage</b></span> to <span class="accent"><b>enemies</b></span> but also <span class="accent"><b>extinguish fires</b></span>.</p>
<p>When fueled with <span class="accent"><b>Petroleum</b></span> or <span class="accent"><b>Oil</b></span>, the <span class="accent"><b>Fluid Cannon</b></span> fires <span class="accent"><b>incendiary bullets</b></span> that <span class="accent"><b>ignite fires</b></span> on impact. It is safer to use outdoors in areas without <span class="accent"><b>Oxygen</b></span>.</p>
